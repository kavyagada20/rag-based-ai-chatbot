# RAG Chatbot 🤖

A simple Retrieval-Augmented Generation (RAG) chatbot built using LangChain and FAISS.

---

## 🚀 Features
- Retrieval-based answering
- Vector database using FAISS
- HuggingFace embeddings
- CLI chatbot

---

## 📦 Setup

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

▶️ Run
python app.py
